# Text Fragment Copier

Firefox extension that adds a context-menu entry for selected text and copies a [scroll-to-text fragment link](https://developer.mozilla.org/en-US/docs/Web/URI/Reference/Fragment/Text_fragments) to the clipboard.

## Install (Temporary)

1. Open `about:debugging#/runtime/this-firefox`.
2. Click `Load Temporary Add-onâ€¦`.
3. Choose the `manifest.json` file from this directory.

## Use

1. Select text on any page.
2. Right-click the selection and choose `Copy Text Fragment Link`.
3. The generated link is placed on the clipboard and can be shared directly.
